# Banking System App 🏦

A simple desktop banking system built in Python, CustomTkinter, & SQLite.

---
## 🚀 Features
- Login System
- Deposit Money
- Payment
- Check Balance
- Session Management
- Logout System
- GUI
- .exe file

---

## 🧰 Tech Used
- Python 🐍
- CustomTkinter
- SQLite
- PyInstaller

---

## ▶️ How to Run
### Option 1 : Run using Python
1. Copy the Repo
   
2. Install CustomTkinter
   ```
   pip install customtkinter
   ```
3. Run the file
   ```
   python main.py
   ```

### Option 2 : Run the executeable file
1. Copy the Repo
2.  Install PyInstaller
   ```
   pip install pyinstaller

   ```


3. Run this command where main.py is there 
   ```
   python -m PyInstaller --onefile --noconsole --icon=ICON/bank.ico --collect-all customtkinter main.py
   ```

4. Open Dist directory and click on the .exe file
