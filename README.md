# Banking System App 🏦

A simple desktop banking system built in Python, CustomTkinter, & SQLite.

---
## 🚀 Features
- Login System
- Deposit Money
- Payment
- Check Balance
- Session Management
- Logout System
- GUI
- .exe file

---

## 🧰 Tech Used
- Python 🐍
- CustomTkinter
- SQLite
- PyInstaller

---

## ▶️ How to Run
### Option 1 : Run using Python
1. Copy the Repo
   
2. Install CustomTkinter
   ```
   pip install customtkinter
   ```
3. Run the file
   ```
   python main.py
   ```

### Option 2 : Run the executeable file
1. Copy the Repo

2. Run the .exe file
   ```
   ./main
   ```
3. Just click the .exe file
